Description

jquery.showhide is jQuery's toggle() on steroids.

jquery.showhide offers some useful features above and beyond what is possible with toggle() alone:

  • Remember the state of the target object in a cookie (via the jQuery Cookie plugin).
  • Change the text of the toggle element (useful if you want the link text to change, e.g. "Show form"/"Hide form").
  • Apply classes to the toggle element to give it plus/minus classes.
  • Auto focus on a form element within the target element (useful if you are showing a hidden form).

To learn more about jquery.showhide, check out the examples below.

Installation

  1. Download and put jQuery and the jQuery Cookie in the <head> of your page:

    <script type="text/javascript" src="jquery.js"></script>
<script type="text/javascript" src="jquery.cookie.js"></script>
  2. Download the latest release of jquery.showhide from Github.

  3. Place jquery.showhide after jQuery and jQuery Cookie:

    <script type="text/javascript" src="jquery.showhide.js"></script>
  4. Use it! See some usage examples and look at the available settings.

Note: It is generally a good practice to concatenate and minify your JavaScript in production environments. Compress you JavaScript here: http://jscompress.com/

Settings

Below is a table of all available settings you can pass to jquery.showhide. All settings are optional.

Option Default Type Description
cookie_expires 365 (one year) Number

Number of days until the cookie expires.

use_cookie must be set to true and the jQuery Cookie plugin must be installed for cookies to work.

cookie_name 'show_target' String

The name of the cookie to store. This must be unique for every element.
default_open true Boolean

Whether or not to have the target element open open by default. This is overridden by any cookies set.
focus_target null String

focus_target takes a String that is evaluated in order to find a field to focus on when showing the target element.

You can use the variables $obj and $target which are jQuery objects of the toggle and target elements to help find a focus_target element, or you can use a jQuery selector. All the code cares about is that focus_target returns a jQuery element.

For example, if you wanted to find the first input box within a form in your target element, you could pass in '$target.find("input")[0]'.

hide_class 'hide' String

The class name to get applied to the toggle link when the target is hidden.
minus_class 'minus' String

The class name to get applied to the link that indicates it is contractable. Can be used to add a "minus" icon to indicate the element can be collapsed.
minus_text null String

The text to show for the toggle element when the target element is shown.
plus_class 'plus' String

The class name to get applied to the link that indicates it is expandable. Can be used to add a "plus" icon to indicate the element can be expanded.
plus_text null String

The text to show for the toggle element when the target element is hidden.
show_class 'show' String

The class name to get applied to the toggle link when the target is shown.
target_obj $obj.next() String

target_obj takes a String that is evaluated in order to find the target element to show/hide.

You can use the variable $obj which is a jQuery object of the toggle element to help find a target_obj element, or you can use a jQuery selector. All the code cares about is that target_obj returns a jQuery element.

For example, if you wanted to find the most recent sibling of the toggle element's parent element, you would pass in '$obj.parent().next()'.

use_cookie false Boolean

If true, use a cookie to store the last chosen state for the target element. If false, do not use a cookie.

Requires the jQuery Cookie plugin to work.
